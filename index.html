✨ Cómo conectar Gemini

Gratis, sin tarjeta de crédito. Solo necesitás una cuenta de Google.

1
Abrí Google AI Studio
Entrá a aistudio.google.com con tu cuenta de Google.
2
Generá una API Key
Hacé clic en Get API key → Create API key. Copiá la clave que empieza con AIza…
3
Pegala en Perfiles
Volvé acá, pegá la clave en el campo API Key de Gemini y guardá los cambios. ¡Listo!
🔒 Tu clave se guarda solo en tu planilla de Google Sheets, no en ningún servidor externo. El plan gratuito de Gemini es suficiente para uso familiar.
